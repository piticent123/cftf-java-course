Sometimes, you'll want to use code you didn't write. There are a lot of reasons you might want to do this. Usually it's because you're trying to do something hard, and someone has already solved the problem for you

Of course, not all code is good, so you should make sure the source is credible, but that's beyond the scope of this task.

To use someone else's code, you just have to import it

An IDE like IntelliJ IDEA often automatically inserts an import statement, but you should still understand how to do it by hand.
